## Teil 3: Akkusativpronomen (4 Punkte)

### Aufgabe 3 (2 Punkte) - 0.5 points each
1. **ihn** (Siehst du den Bus? - Ja, ich sehe ihn.)
2. **sie** (Kennst du Maria? - Ja, ich kenne sie.)
3. **es** (Brauchst du das Auto? - Ja, ich brauche es.)
4. **mich** (Liebst du mich?)

### Aufgabe 4 (2 Punkte) - 0.5 points each
1. Ich sehe **ihn**.
2. Wir suchen **sie**.
3. Er braucht **es**.
4. Sie kennt **uns** / **euch**.

## Teil 5: Wegbeschreibung (4 Punkte)

### Aufgabe 6 (2 Punkte) - 0.67 points each (rounded)
A: Entschuldigung, wo ist der Bahnhof?

B: Gehen Sie **geradeaus** (straight ahead), dann **nach links** (to the left). Der Bahnhof ist **rechts** / **auf der rechten Seite** (on the right).

### Aufgabe 7 (2 Punkte) - 0.5 points each
1. geradeaus → ↑ (up arrow)
2. nach links → ← (left arrow)
3. nach rechts → → (right arrow)
4. zurück → ↓ (down arrow)

**Note:** Accept any reasonable arrow representation of directions.
