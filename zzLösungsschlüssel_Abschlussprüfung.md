## Teil 3: Verben Konjugation (15 Punkte)

### Aufgabe 3.1 (6 Punkte) - 1 point each
| Pronoun | sein |
|---------|------|
| ich | **bin** |
| du | **bist** |
| er/sie/es | **ist** |
| wir | **sind** |
| ihr | **seid** |
| sie/Sie | **sind** |

### Aufgabe 3.2 (3 Punkte) - 1 point each
| Pronoun | haben |
|---------|-------|
| ich | **habe** |
| du | **hast** |
| er/sie/es | **hat** |

### Aufgabe 3.3 (6 Punkte) - 1 point each

3. Wir **essen** Pizza.
4. Sie **fährt** mit dem Bus.
