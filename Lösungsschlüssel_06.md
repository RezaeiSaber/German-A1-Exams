## Teil 1: Stadt - Wortschatz (4 Punkte)

### Aufgabe 1 (2 Punkte) - 0.5 points each
1. **der** Bahnhof
2. **die** Schule
3. **das** Krankenhaus
4. **das** Kino

### Aufgabe 2 (2 Punkte) - 0.5 points each
1. **Bank**: ○ der ☑ **die** ○ das
2. **Restaurant**: ○ der ○ die ☑ **das**
3. **U-Bahn**: ○ der ☑ **die** ○ das
4. **S-Bahn**: ○ der ☑ **die** ○ das

## Teil 2: Adjektive (3 Punkte)

### Aufgabe 3 (3 Punkte) - 1 point each
1. schön ≠ **hässlich**
2. teuer ≠ **billig**
3. laut ≠ **ruhig** (or: leise)

---

## Teil 3: Dativ - Statische Position (5 Punkte)

### Aufgabe 4 (3 Punkte) - 1 point each
1. Ich wohne **in der** Stadt.
2. Er lebt **auf dem** Land.
3. Das Geschäft ist **am** Bahnhof.

### Aufgabe 5 (2 Punkte) - 0.5 points each
1. Ich wohne **(in der** / in die) Stadt.
2. Sie lebt **(auf dem** / auf den) Land.
3. Die Wohnung ist **(im** / in der) Zentrum.
4. Er ist **(am** / an der) Bahnhof.

---

## Teil 4: "Es gibt" + Akkusativ (4 Punkte)

### Aufgabe 6 (4 Punkte) - 1 point each
1. Es gibt **einen** Kindergarten.
2. Es gibt **ein** Kino.
3. Es gibt **eine** Kirche.
4. Es gibt viele **Restaurants**.

## Teil 5: Zimmer und Möbel (4 Punkte)

### Aufgabe 7 (2 Punkte) - 0.5 points each
**1 =** **D** (living room)  
**2 =** **B** (bedroom)  
**3 =** **C** (kitchen)  
**4 =** **A** (bathroom)

### Aufgabe 8 (2 Punkte) - 0.5 points each
1. sofa/couch = **die Couch** (or: das Sofa)
2. bed = **das Bett**
3. table = **der Tisch**
4. chair = **der Stuhl**
